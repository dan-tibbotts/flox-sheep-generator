FLOX - Sheep Generator

Welcome to FLOX. The purpose of this application is to generate a flock of sheep.
Each sheep is represented by an oval.

How to Generate a Flock

  1. Select the total amount of sheep you want from 1 - 50.
  2. Select the weight range (size of the oval) for the sheep between 15-30.
    Each sheep will have a weight between the min and max value.
  3. Select the speed each sheep will move.
    Select a Range From 1 to 5 (1 = slow, 5 = fast)
  4. Click anywhere on the canvas (field) to add more sheep on the fly

Other Settings

  • The mouse co-ordinates (X and Y) position will be displayed in the settings
  • Change the background to suit for flock by selecting a new "BG Color"
